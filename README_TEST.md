# Selenium TestNG Homework – Test Instructions

## Як запустити тести

### Через термінал (Maven)
mvn clean test

### Через IntelliJ IDEA
Run 'SeleniumTestngTest' у редакторі

### Через GitHub Webpage
Створи нову гілку → Upload files → Create Pull Request
